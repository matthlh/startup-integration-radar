When the chat is getting long, run this exact command:

/compact Focus on Integration Scout project state, current branch/commit, files changed, API routes, CLI commands, frontend components, tests run, unresolved bugs, and exact next task. Do not preserve repetitive discussion or old brainstorming.

After compaction, run:

/save-compact

Use this after big tasks, before switching tasks, or every ~30-50 messages.
